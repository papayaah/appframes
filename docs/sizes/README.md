## Last Updated

**November 24, 2024**

All documentation reflects the requirements as of this date. Store requirements may change over time.

## Quick Start

### For Apple App Store Submission
1. Read [apple-app-store.md](./apple-app-store.md)
2. Use exact dimensions specified
3. Export from ScreensStudio with pre-configured sizes

### For Google Play Store Submission
1. Read [google-play-store.md](./google-play-store.md)
2. Choose dimensions within allowed ranges
3. Ensure 16:9 or 9:16 aspect ratio
4. Use recommended common resolutions

## Key Differences

| Aspect | Apple | Google |
|--------|-------|--------|
| Dimensions | Exact sizes required | Flexible ranges |
| Aspect Ratio | Device-specific | 16:9 or 9:16 |
| Flexibility | Low | High |
| File Size Limit | Not specified | 8-15 MB |

## Verification

Always verify current requirements before submission:
- **Apple:** [App Store Connect](https://appstoreconnect.apple.com)
- **Google:** [Play Console](https://play.google.com/console)
